** Sobre carregaMapa
- Toda vez que a opção de "Iniciar um novo" jogo for chamada, a posição do player (e futuramente dos inimigos) deve ser resetada (com base no mapa);

*** O flow sera:
1. carregaMapa() popula a matriz de char; Isso só deve ocorrer uma unica vez, ficando a matriz então salva em memória e contendo todas as informações sobre o mapa iniciais
2. Para o player (e para inimigos e demais entidades), suas posições iniciais são atribuídas a seus respectivos structs, podendo variar livremente. Para isso, uma pequena função será feita cuja responsabilidade é fazer essa atribuição das posições nos structs a partir da matriz de caracteres.
